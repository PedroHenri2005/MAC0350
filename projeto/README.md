# Nihongo Flashcards (日本語フラッシュカード）
## Ideia do projeto
A intenção desse projeto será criar um sistema de flashcards direcionados para o aprendizado da língua japonesa. Flashcards são simplesmente cartões que possuem na parte da frente um conceito qualquer(nesse caso, uma palavra em japonês), e na parte do verso, a explicação sobre ele(nesse caso, a tradução da palavra). É um método muito conhecido para a retenção de conhecimento nos estudos.
## Front-end
A ideia é que inicialmente o card mostre a palavra não traduzida, e então será necessária a interatividade do usuário(clicando ou passando o mouse por cima do card) para fazer com que a tradução apareça quando ele achar conveniente. 
## Back-end 
Além disso, deverá ser possível para o usuário adicionar e retirar palavras do seu deck(conjunto de flashcards) para manipulá-lo como quiser, o que exigirá manipular uma base de dados contendo essas palavras.
